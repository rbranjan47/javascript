                                    JavaScript Keyword

var --> Declares the variable
let --> Declares block variable, scope of let keyword is BLOCK level
const --> Declares the block constant, scope of const is BLock level
if --> Mark a block of code to be executed on a conditions
switch --> Mark a block of code to be executed in a different cases
for --> Marks a block of code to be executed in a loop
function --> Declares a function
return --> exits from a function
try --> Implements error handling to a block of statements
